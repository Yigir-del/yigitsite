## 2. Motto (bu siteyi yaparkenki ruh)

Site bir klasik CV vitrini değil. Motto kabaca şöyle:

> **İzler bırakılır. Bazıları senindir. Hiçlikle şaka yapılır. Kural koyan kraldır — ama herkes uçabilir.**

Daha net maddeler:

1. **İz bırakmak** — Ana metafor. Not, düşünce, proje, easter egg: hepsi “iz”.
2. **Sahip olmak / paylaşmak** — “Bazıları senin olabilir.” Ziyaretçi de iz bırakır (not defteri).
3. **Absürt ciddiyet** — Felsefe + dilenci + Spotify diktatörlüğü yan yana durur; çelişki kasıtlıdır.
4. **Kral = ziyaretçi / site sahibi hissi** — Dilenci kralla konuşur. Müzik: “mekan benim, kurallar benim.”
5. **Hiçlik / boşluk** — Sağ tık menüsü, idle sözler, footer: boşluğa bakmak, hiçlikten merhaba.
6. **Kontrollü kaos** — Şeyler uçar, fırlatılır, ters döner; ama masaüstü zengin, mobil hafif kalır.
7. **Türkçe öncelikli ses** — Arayüz ve diyalog Türkçe; SEO description İngilizce olabilir.

Kısa AI özeti:

> “Karanlık yıldızlı bir alanda, glitch’li tipografiyle konuşan, ziyaretçiyi kral yapan, iz bıraktıran, absürt ama samimi bir kişisel portfolyo.”

## 3. Görsel / atmosfer jargon

- **MURYOKUSHO** — Tek domain / tema. Soğuk lacivert-gri, soluk yıldız, cam (glass), hafif glow. Mor / krem / “AI purple gradient” yok.
- **Glitch başlıklar** — `glitch` class; metin biraz bozulmuş ama okunur.
- **Handwriting** — Caveat ile küçük fısıltılar: `(aşağıda bir defter var)`.
- **Uçan varlıklar** — İkonlar ve karakterler ekranın *dışından* girer, kenardan kenara uçar; ortada “doğup yok olmaz”.
- **World-flip** — Footer’a çok tıklayınca dünya 180° döner; arka plan sabit kalır.
- **Mobil kuralı** — Ağır WebGL, Lenis, uçan kaos, blur, drag-fırlatma mobilde kesilir veya sadeleşir. Desktop bozulmaz.

## 4. Ana sayfa copy (kanon cümleler)

**Hero**

- “İzler burada kalır.”
- “Bazıları senin olabilir.”
- “(aşağıda bir defter var)”

**Not duvarı**

- Başlık: “Not Defterim”
- Boşken: “Henüz iz yok.”
- Kalem tooltip: “Bırakmak istediğin bir iz var mı?”
- Anonim yazar varsayılanı: “Gizemli Yabancı”
- Notlar sağa-sola saçılır, PC’de fırlatılır, bir süre sonra yavaşça yerine döner; mobilde fırlatma yok.

**Müzik kutusu**

- Panel adı: “Senin Frekansın”
- Arama troll’ü: istek parçayı reddeder (DJ / kral tonu).
- Küçült (–) = müzik devam; ✕ = durdur/kapat.
- Örnek sözler: “Burada demokrasi yok…”, “DJ benim…”, “Spotify Premium’umu sen mi ödüyorsun?”

**Stüdyo**

- “Stüdyom” / “Ortaya Karışık”
- Fotoğraflar: seç → önizle → onayla → yerleştir

**Footer havası**

- “Şu an CSS biliyormuş gibi yapıyorum.”
- “Kahve ve kötü kararlarla çalışıyor.”
- “404 uyku bulunamadı.”
- “Hiçlikten merhabalar.”
- Sonuna kadar tıklama zinciri → world-flip

**Sağ tık (FakeMenu)**

- “Gerçekliği inceliyorum…”
- “Aslında her şey bir hiçlikten ibaret.”

## 5. Karakterler (uçan pantheon)

### Dilenci
- Sinirli, aç, krala dilenen yüz.
- Çoğunlukla **kendi işi** (kraldan bağış).
- Nadiren bilgeye laf sokar; bilge laf edince bazen cevap verir.
- Ton: kaba, komik, gururlu dilenci.

### Bilge
- Laf sokarak öğüt veren sakallı bilge.
- Çoğunlukla **özlü söz**; nadiren dilenciye iğne.
- Ton: ağırbaşlı ama sivri.

### Kalem
- Nota iz bırakma aracı.
- Soft, davetkâr.

### Müzik
- Site DJ’i / frekans bekçisi.
- Demokrasi yok; arama alaycı.

### Sosyal drifter’lar
- GitHub / LinkedIn ikonları kenardan uçar.
- Tooltip: “GitHub'a uğra.” / “LinkedIn tarafı.”

### Chaos
- UFO/hayalet, sahte popup, achievement, idle felsefe.
- Idle: “Boşluğa ne kadar uzun süre bakarsan…”

**Diyalog kuralı:** Karakterler *çoğunlukla kendi işine bakar*; birbirlerine laf etmeleri seyrek (~%15). Random konuşurlar ama bilge↔dilenci jab’lerinde eşleşmiş cevaplar vardır.

## 8. Teknik gerçekler (AI’nın bilmesi gereken)

- Stack: React 19 + Vite + React Router, Framer Motion, GSAP (desktop), Three.js yıldız alanı (desktop only)
- Backend: Vercel API + Postgres + Blob (notlar, stüdyo foto)
- Admin: `localStorage yigit_admin === 'true'`
- Performans kapısı: `useIsMobilePerf` (~860px veya coarse pointer)
- Canonical / marka URL: yalnızca `https://yigitaltuntas.me`
- Push alışkanlığı: iş bitince `main`’e commit + push
